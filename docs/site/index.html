Quick Start

Hit the ground running with the core SU7 + optics workflow.

  1. 1. Load the canonical preset

    Launch the studio, choose Presets → Phase 12 Launch, and explore the curated Kuramoto + optics configuration.

  2. 2. Tweak the Kuramoto driver

    Adjust K₀ and α in the Dynamics panel to watch coupling synchronization shift in real time.

  3. 3. Export an optical render

    Head to Export → Thin Element Stack and generate amplitude + phase passes for downstream compositing.
